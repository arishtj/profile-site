# Arisht Jain Portfolio Site

Interactive portfolio website focused on Cloud Engineering and DevOps.

## What this site includes

- Premium dark UI with responsive layout
- Hero section with strong Cloud/DevOps positioning
- Animated impact counters
- Experience timeline (UA Finance, KM.ON, Fleet)
- Cloud/platform stack section with logos
- Certification progress bars
- Contact section with email + LinkedIn

## Project structure

- `index.html` — page structure and content
- `styles.css` — visual system, layout, and responsive styles
- `script.js` — interactions and animations
- `assets/aws.svg` — local AWS logo asset

## Run locally

```bash
cd arisht-site
python3 -m http.server 8080
```

Then open:

- `http://localhost:8080`

## Notes

- This is a static site (no backend required).
- Best edited directly in `index.html`, `styles.css`, and `script.js`.
